# Cloud-Based E-Commerce Application with DevOps Pipeline

## Overview

This project is a cloud-native e-commerce application designed to demonstrate modern DevOps practices. It includes a web frontend, backend APIs, and a MySQL database. The application will be containerized using Docker, deployed on AWS with Kubernetes, and automated through a Jenkins CI/CD pipeline.

## Features

* Browse available products
* Add and remove items from the shopping cart
* Place customer orders
* Dockerized frontend and backend services
* Automated CI/CD with Jenkins
* Infrastructure provisioning using Terraform
* Deployment on AWS EKS
* Monitoring with Prometheus, Grafana, and CloudWatch

## Technologies

* React
* Node.js & Express
* MySQL
* Docker
* Kubernetes
* Jenkins
* Terraform
* AWS
* Prometheus & Grafana
